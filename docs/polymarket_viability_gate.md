# Polymarket 수익 가능성 판정 기준 (Viability Gate)

Last updated: 2026-02-17 (KST)
Owner: seoseo-ai

## 1) 목적
폴리마켓 참여 시, 감각/낙관이 아닌 데이터 기반으로
"수익화 진행 가능 여부"를 판정하기 위한 최소 기준입니다.

---

## 2) 데이터 최소 요건 (미달 시 판정 보류)
- 실거래 표본: 최소 50건 (권장 100건)
- 관찰 기간: 최소 6주
- 로그 누락률: 5% 이하

위 3개 중 하나라도 미달이면 판정은 `보류`입니다.

---

## 3) 합격 컷라인 (전부 충족 필요)

### A. 기대값(EV)
- 거래당 기대값(EV) > 0
- 최근 4주 rolling EV > 0

### B. 수익 안정성
- 주간 수익 플러스 주 비율 60% 이상

### C. 리스크 통제
- 최대낙폭(MDD) -15% 이내
- 일손실 상한 위반 0회

### D. 운영 품질
- 규칙 위반률 5% 이하
- 근거 없는 진입 0회

---

## 4) 경고 구간 (발생 시 수익화 금지)
- 최근 20트레이드 EV 음수
- 2주 연속 손실
- MDD -15% 초과
- 손실 직후 배팅 사이즈 증가(복구매매 징후)

---

## 5) 최종 판정 등급
- `PASS`  : 소액 수익화 진행 가능
- `HOLD`  : 검증 계속 (수익화 보류)
- `FAIL`  : 수익화 불가 (전략/운영 재설계 필요)

판정 규칙:
- A/B/C/D 전부 충족 + 경고 구간 없음 => `PASS`
- 최소요건 충족했으나 일부 미달 => `HOLD`
- 최소요건 미달 또는 경고 구간 발생 => `FAIL`

---

## 6) 운영 원칙
- 본 문서 기준을 충족하기 전에는 "규모 확대" 금지
- 결과 보고는 반드시 로그/증빙 링크 포함
- 월간 목표는 금액보다 "프로세스 안정성 + EV 검증" 우선
