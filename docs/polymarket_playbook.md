# Polymarket 운영규정(1p)

## 목표 (현실형)
- 기간: 1개월
- 목표 수익률: **+8% ~ +12%**
- 총 원금 한도: $100
- 방식: 반자동(B) — 에이전트 제안 + 사람 실행

## 가드레일(확정)
- 1회 베팅 상한: $20
- 일일 손실 상한: $10

## 운영 원칙
- 로그 없으면 무효: 모든 트레이드는 `logs/polymarket_trades.csv`에 기록
- 리뷰 주기: 최소 주 1회(요약/회고)
- 에이전트 팀 업무는 **기본 서서 단독 처리** (분업이 명확히 유리할 때만 예외 적용)

## 로그 작성 최소 기준
- entry_reason / exit_reason는 반드시 작성
- stop_rule에 해당 트레이드에 적용된 제한을 명시
