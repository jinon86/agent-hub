# Polymarket 24h Contest Plan (v1)

## 목적
- 짧은 기간(24h) 동안 규칙 기반 의사결정과 리스크 통제를 검증
- 수익률뿐 아니라 손실관리, 규칙 준수 역량을 함께 평가

## 콘테스트 기본 규칙
- 참가자: 서서 / 방통 / 소교
- 시작금: 각 $100
- 기간: 24시간
- 승자 기준: (최종자산 - 시작금) / 시작금 * 100
- 동률 시 우선순위:
  1) MDD(최대낙폭) 낮은 사람
  2) 규칙 위반 0건

## 필수 리스크 규칙
- 1회 포지션 최대: 자산의 20%
- 동시 포지션 최대: 3개
- 일손실 -15% 도달 시 해당 참가자 당일 종료
- 금지 항목:
  - 감정 진입
  - 손실 복구용 물타기/복수매매
  - 근거 없는 추격매수

## 실행/보고 표준
- 진입 시:
  `IN PROGRESS / <이름> / <시장> / 진입가 / 근거링크`
- 청산 시:
  `DONE / <이름> / <시장> / 청산가 / 손익%`
- 장애/차단 시:
  `BLOCKED / <이름> / <이슈 1줄> / <필요지원>`

## 측정 KPI
1) 순수익률(%)
2) MDD
3) 규칙 위반 횟수

## 운영 역할
- 대표: 최종 승인/승자 확정
- 서서(팀장): 규칙 집행, 집계, 분쟁 판정
- 방통/소교: 거래 실행, 실시간 보고, 증빙 기록

## 시작 전 체크리스트 (5분)
1) 참가자별 시작 잔고 캡처
2) 시작/종료 시각 고정
3) 보고 포맷 공지
4) 규칙 위반 페널티 확정(권장: 실격 또는 -10%p)
5) 증빙 저장 위치 확정

## 종료 산출물
- 참가자별 최종 수익률 표
- MDD 및 규칙 위반 건수
- 상위 전략 요약(무엇이 먹혔는지)
- 개선사항(다음 회차 룰 조정)
